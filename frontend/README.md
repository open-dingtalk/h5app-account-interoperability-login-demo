# 钉钉SSO登陆绑定Demo
***
## Getting Started
***
### 克隆代码到本地
git clone

`git clone xxxxxxxxxx`
### 安装依赖
install dependencies

`$ npm install`
### 开发
demo必须在钉钉环境下才可以获取到钉钉用户信息，在浏览器中不支持获取。
***
## 使用开发者工具
https://www.jetbrains.com/idea/download/#section=windows

打开idea，导入项目

## 项目启动
`$ npm run serve`

## 项目打包
`$ npm run build`
## 项目地址
http://localhost:port/#/
## 配置的微应用地址
http://部署的服务器的ip:port/#/
(port是nginx配置的反向代理的前端项目地址)
